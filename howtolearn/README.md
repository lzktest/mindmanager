## 个人总结

学习过程主要是4个步骤：  获取关键信息-----根据信息的背景、因果等关系进行拓展------将信息与已熟知的事物或者只是相联系（便于记忆和理解）------建立知识模型（便于快速理解新知识）------将知识应用于生活（加深理解）     随时修正模型和联系
